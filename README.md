## 1. Introduction to Linux

- What is Linux?
- History and development
- Open-source philosophy
- The role of the kernel

## 2. Linux Distributions

- Popular distros: Ubuntu, Debian, CentOS, Fedora, Arch
- Package managers:
  - APT (Debian/Ubuntu)
  - YUM/DNF (RedHat/CentOS/Fedora)
  - Pacman (Arch)

## 3. Basic Shell Commands

- `pwd`, `cd`, `ls`, `cp`, `mv`, `rm`, `mkdir`, `rmdir`
- `man`, `clear`, `echo`, `cat`, `more`, `less`
- Command chaining: `|`, `>`, `>>`, `&&`, `;`

## 4. Users and Permissions

- `useradd`, `passwd`, `usermod`, `groupadd`
- File ownership: `chown`, `chgrp`
- File permissions: `chmod`
- Understanding `rwx` and permission levels

## 5. File System Hierarchy

- Common directories:
  - `/bin`, `/etc`, `/home`, `/var`, `/usr`, `/tmp`, `/root`
- Mounting devices: `mount`, `umount`

## 6. Text Editors

- `nano`, `vim`, `gedit`
- Basic editing commands in `vim`
- Saving and exiting files

## 7. Package Management

- APT commands: `apt update`, `apt install`, `apt remove`
- YUM/DNF commands: `dnf install`, `dnf remove`
- Searching and installing packages

## 8. Networking Basics

- Checking network: `ip a`, `ping`, `ifconfig`, `netstat`
- Hostname configuration: `hostname`, `hostnamectl`
- SSH: `ssh user@host`

## 9. System Services

- Systemd: `systemctl start`, `stop`, `restart`, `status`
- Enabling and disabling services
- Checking logs: `journalctl`

## 10. Basic Scripting

- Writing a shell script
- `#!/bin/bash` header
- Variables, loops, and conditionals
- Running scripts: `bash script.sh` or `./script.sh`
